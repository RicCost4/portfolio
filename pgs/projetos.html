Analisador de Números

Codigo no Repositorio - click aqui

Esse projeto foi desenvolvido como estudo de manipulação de DOM em JS. O HTML e o CSS são simples e foi usado como modelo só para visualizar o funcionamento. Essa atividade mostra a captura dos números digitados e quando o usuário quiser finalizar, irá mostrar uns dados sobre os números digitados, ele valida certas ações do usuário feita incorretamente 😂 ex: 1, 2, 5, 10, 45 → FINALIZAR = "Ao todo, temos 5 números cadastrados; O maior valor adicionado foi 45; O menor valor adicionado foi 1; somando todos os valores, temos 63; A média dos valores digitados é 12.6."

O código em JS começa selecionando o HTML e armazenando numa variável e declaramos um array para armazena os números captados pelo input (usei o let para declarar as variáveis, pois preferi com exemplo de declaração em bloco, e não em escopo).

Agora iremos criar várias funções para administrar nossos números e mostrar o resultado no nosso HTML:

  • Função para validar número;
  • Função para ver se o número e repetido;
  • Função para adicionar o número na lista validando se é valido ou se não está já na lista e mostrando ele na caixa criada no HTML quando clicamos em adicionar número;
  • Função para mostrar os dados específicos da lista e dos números adicionado quando clicamos em validar;
